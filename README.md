# Product Feature Usage & Adoption Analysis

During this project, I wanted to understand how students actually use Google Docs and Google Drive in their daily academic work. I collected survey responses, analyzed user behavior, and built an interactive dashboard to explore feature adoption, awareness levels, engagement, and usage patterns.

Using Python, Pandas, Streamlit, and Plotly, I transformed raw survey data into meaningful visual insights through KPI metrics, charts, cohort analysis, and funnel analysis. This project helped me gain practical experience in data analysis, dashboard development, and product analytics.
